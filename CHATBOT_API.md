# TRAVAUX IA - Chatbot API Documentation

This document describes the open text-in/text-out API endpoint provided by the Devis Generation API, powered by the Scaleway `gpt-oss-120b` model.

The chatbot acts as the **Core Assistant Intelligence** for TRAVAUX IA, with full context on the application's React/Supabase architecture, the RAG logic for quotes, and the two-stage generation pipeline.

## Endpoint Overview

- **URL:** `/api/v1/chat`
- **Method:** `POST`
- **Content-Type:** `application/json`
- **Authentication:** None (Open API)

## Request Structure

The endpoint expects a simple JSON object containing the `text` you wish to send to the chatbot.

```json
{
  "text": "Your message here..."
}
```

## Response Structure

The endpoint responds with a JSON object containing the `text` generated by the AI assistant.

```json
{
  "text": "The AI's generated response..."
}
```

## Example Usage

### Request (curl)

```bash
curl -X POST "http://localhost:8000/api/v1/chat" \
     -H "Content-Type: application/json" \
     -d '{"text": "Can you explain how the EnhancedAIDevisService prevents combining multiple material typologies in a single quote line?"}'
```

### Success Response (HTTP 200)

```json
{
  "text": "To prevent combining multiple material typologies, the EnhancedAIDevisService utilizes the PRESTATION_ANALYSIS_PROMPT which strictly enforces the '1 Typology = 1 Line' rule. This means if a user requests both 'Tuiles mécaniques' and 'Ardoises', the model must generate two separate execution lines rather than combining them into a single line..."
}
```

## Error Codes

- `400 Bad Request`: If the `text` payload is missing or empty.
- `422 Unprocessable Entity`: If the request JSON structure is invalid.
- `503 Service Unavailable`: If the Scaleway Generative API cannot be reached or returns an error.
- `500 Internal Server Error`: For any other unexpected server-side failures.

## AI Model Configuration

- **Model:** `gpt-oss-120b` (Scaleway)
- **Max Tokens:** 8192
- **Reasoning Effort:** Medium
- **Temperature:** 0.7 
- **System Prompt:** Embedded deeply with the "Core Assistant Intelligence" context, ensuring architecture-aware responses.
